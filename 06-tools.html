Outils

W3C

Le W3C propose une liste d'outils pour évaluer l'accessibilité des sites web. Ces outils peuvent être utilisés pour vérifier la conformité aux normes d'accessibilité.

Liste des outils d'évaluation de l'accessibilité du W3C

Il propose également un outil permettant de réaliser un rapport d'accessibilité en suivant les critères du WCAG :

Outil de rapport d'accessibilité du W3C

Cependant, cet outil ne réalise pas de vérification automatique, il est donc nécessaire de le compléter manuellement.

Tanaguru

https://www.accessibilitychecker.org/

Chrome dev tools

  • https://developer.chrome.com/blog/full-accessibility-tree

Pour vérifier le contrast des couleurs, vous pouvez utiliser l'outil intégré dans les DevTools de Chrome. Ouvrez les DevTools, allez dans l'onglet "Elements", sélectionnez un élément, puis cliquez sur l'icône de contraste des couleurs dans la barre latérale droite.

Firefox dev tools

L'outil d'accessibilité de Firefox est intégré dans un onglet à part entière du DevTools.
