Pré-requis

Balises HTML sémantiques

Avant de commencer à traiter l'accessibilité en programmation, nous allons revoir quelques notions importantes en HTML.

HTML - Attributs utiles

Quelques attributs HTML importants pour l'accessibilité :

  • lang : Définit la langue du document ou d'un élément. 
    <html lang="fr">
  • alt : Texte alternatif pour les images. 
    <img src="photo.jpg" alt="Description de l'image">
  • tabindex : Contrôle l'ordre de navigation au clavier.
    • tabindex="0" : Rend l'élément focusable dans l'ordre naturel du DOM.
    • tabindex="-1" : Focusable uniquement via JavaScript (element.focus()), pas par la touche Tab.
    • tabindex="1+" : À éviter. Les valeurs positives modifient l'ordre de navigation et créent une expérience imprévisible pour les utilisateurs clavier. Il faut privilégier l'ordre naturel du DOM.

HTML - Formulaires accessibles

Associer correctement les labels aux champs de formulaire est essentiel : 

<!-- Méthode 1 : Attribut for/id -->
<label for="email">Email</label>
<input type="email" id="email" name="email">

<!-- Méthode 2 : Label englobant -->
<label>
    Email
    <input type="email" name="email">
</label>

<!-- Indiquer les champs obligatoires -->
<label for="name">Nom</label>
<input type="text" id="name" required>

HTML - Attribut autocomplete

L'attribut autocomplete permet aux navigateurs et aux technologies d'assistance d'identifier la nature des champs de formulaire et de les pré-remplir automatiquement. 

<form>
    <label for="name">Nom</label>
    <input type="text" id="name" autocomplete="name">

    <label for="email">Email</label>
    <input type="email" id="email" autocomplete="email">

    <label for="tel">Téléphone</label>
    <input type="tel" id="tel" autocomplete="tel">

    <label for="address">Adresse</label>
    <input type="text" id="address" autocomplete="street-address">
</form>

HTML - Structure sémantique

Utiliser les balises HTML sémantiques améliore l'accessibilité et le référencement : 

<header>
    <nav>...</nav>
</header>

<main>
    <article>
        <h1>Titre principal</h1>
        <section>
            <h2>Sous-titre</h2>
            <p>Contenu...</p>
        </section>
    </article>
    <aside>Contenu secondaire</aside>
</main>

<footer>...</footer>
